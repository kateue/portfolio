StudyUp

Web Application Interface Design and System Analysis

Project Overview

StudyUp's goal is to unify UVic's existing study space booking systems under one centralized system that is both filterable and easy to use for students. This goal was motivated by the lack of a centralized study space booking system, as most of the current UVic systems are fragmented across buildings, causing confusion and wasting time for students.

To approach this issue, StudyUp intends to provide UVic students with our developed centralized website, StudyUp, for them to comfortably book study space rooms through.



Project Description

Current UVic Study-Room Booking Site





Gen8 investigated issues with UVic's study space booking systems, discovering their complexity and dispersion across multiple websites. This aligns with UVic's concern, making it challenging for over twenty thousand enrolled students to locate campus study spaces efficiently.

Pain Points



Complicated and Dispersed Systems: The existing study space booking systems at UVic are complex and dispersed across various websites. Each site has an arbitrary number of buildings, making it difficult for students to navigate.

Lack of Filtering and Descriptions: Insufficient filtering options and room descriptions hinder the ability to find suitable study spaces. Certain sites, such as the UVic Library Web App, lack these essential features.

Functional Issues: Certain sites experience irregular malfunctions during the room booking process. Asynchronous delays occur when browsing open availability, impacting the user experience.

Proposed Solutions



User-Friendly Interface: Optimized for ease of use, catering to students with varying levels of technological literacy.

Filtering System: Enables users to browse available listings with synchronous feedback for a more streamlined experience.

User Data Collection and Analysis: Gathers and analyzes user data to maintain and upgrade the application over time.
